Administration - Sujets

Vous devez définir des sujets pour pouvoir publier vos articles.
C'est un composant essentiel du système. Le système vous permet d'ajouter, de supprimer ou de modifier les sujets.
Chaque sujet doit avoir un intitulé, une description, éventuellement une image et un ou plusieurs administrateurs.

Ajouter un sujet :

  • Intitulé : c'est l'identifiant qui sera utilisé par NPDS. Saisissez le en minuscules et sans espace et sans accent
  • Texte : c'est le texte qui sera affiché. Vous pouvez utiliser les majuscules/minuscules et les espaces.(maxi 40 caractères)
  • Image : NPDS permet de ne pas avoir d'image mais si vous souhaitez en affecter une, saisissez dans cette zone le nom du fichier par exemple : npds.gif. Ce fichier doit se trouver dans le répertoire assets/images/topics à la racine du site ou dans le dossier de votre thème.
  • Administrateur : NPDS permet de définir un ou plusieurs administrateurs pour un sujet.
    Si cette zone est laissée vide, seuls les superadmins pourront publier dans ce sujet.
    Si cette zone contient le nom d'un ou plusieurs administrateurs il faut séparer les noms par des virgules.

Editer un sujet :

Cliquez simplement sur l'image ou le texte dans le tableau récapitulatif. Dans la page suivante vous pourrez modifier l'intitulé, la description et l'image.

Vous pouvez aussi supprimer le sujet mais rappelez-vous que cela supprime aussi tous les articles rattachés.
Un message de confirmation doit être renseigné.

Sur cette page vous pouvez aussi affecter des liens web en rapport avec le sujet.
Le lien relatif sera affiché avec les articles du sujet.
